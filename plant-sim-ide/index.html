🌱 Plant Sim IDE — paste sketch + diagram, click Run, watch your plant.

How this IDE works

What it does: parses the constants from your sketch.ino and feeds them to a visual simulation of the plant + tank + pump + nozzle scene. Edit the constants, hit ▶ Run, and watch how your changes affect the watering behaviour.

Constants the simulator reads

  • DRY_THRESHOLD — moisture raw value at/above which soil is "dry"
  • WET_FLOOR — raw value at/below which soil shows 100%
  • PUMP_BURST_MS — how long the pump runs per burst
  • SOAK_WAIT_MS — wait time after a burst before re-checking
  • PUMP_COOLDOWN_MS — minimum gap between pump bursts
  • SAMPLE_INTERVAL_MS — how often the soil sensor is read
  • MAX_PUMPS_PER_DAY — hard daily watering cap
  • PUMP_ACTIVE_LOW — relay polarity (cosmetic in the sim)

Install as a desktop app

Look for the ⬇ Install app button in the top bar (appears in Chromium browsers when the IDE is hosted over HTTP/HTTPS). The IDE will install as a standalone window with its own icon and work fully offline.

What this is not

Not a real Arduino emulator. C++ is not compiled. The IDE extracts tuning constants and feeds them to a JavaScript port of the same state machine — accurate for testing thresholds and timings, not for testing arbitrary new C++ logic.

Hot keys

  • Ctrl + Enter — Run
  • Ctrl + S — Save sketch as .ino
  • Monaco editor shortcuts: Ctrl + / comment, F1 command palette, Alt + ↑/↓ move line
